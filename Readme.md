# 🤖 APIA Expert: مساعد الاستثمار الفلاحي الذكي

**APIA Expert** هو تطبيق ويب متطور يعتمد على تقنيات الذكاء الاصطناعي (Generative AI)، مصمم خصيصاً لمرافقة المستثمرين في قطاع الفلاحة بتونس. يعمل النظام كخبير رقمي لوكالة النهوض بالاستثمارات الفلاحية (**APIA**)، حيث يقدم إجابات دقيقة ومحدثة بناءً على الوثائق الرسمية المنشورة.

---

## 🌟 المميزات الرئيسية
*   **إجابات موثقة:** يعتمد النظام على تقنية **RAG** (Retrieval-Augmented Generation) لتقديم معلومات مستقاة مباشرة من ملفات الـ PDF الخاصة بالوكالة لضمان عدم تغيير الحقائق أو الأرقام.
*   **واجهة زجاجية (Glassmorphism):** تصميم عصري بواجهة شفافة تتناسب مع الهوية البصرية للوكالة.
*   **دعم كامل للغة العربية:** واجهة مستخدم محسنة للغة العربية (RTL) مع تنسيق دقيق للنصوص والجداول.
*   **سرعة استجابة فائقة:** مدعوم بمحرك **Gemini 2.5 Flash** لضمان تجربة محادثة سلسة وسريعة.

---

## 🚀 التقنيات المستخدمة (Tech Stack)

*   **الذكاء الاصطناعي:** [Google Gemini 2.5 Flash API](https://aistudio.google.com/).
*   **إطار العمل:** [Streamlit](https://streamlit.io/).
*   **لغة البرمجة:** Python 3.11.
*   **التنسيق:** Custom CSS & HTML Integration (لتخصيص صندوق السؤال والبطاقة الترحيبية).
*   **إدارة النسخ:** GitHub.

---

## 🛠️ مراحل الإنجاز (Project Milestones)

### 1. مرحلة التأسيس (Infrastructure)
*   إعداد بيئة العمل على **Google AI Studio** وضبط معايير السلامة وطول المخرجات لضمان دقة الاستجابة.
*   هيكلة قاعدة بيانات المعرفة عبر جمع ومعالجة تقارير ووثائق الوكالة الرسمية السنوية.

### 2. التطوير البرمجي (Backend)
*   تطوير منطق استرجاع المعلومات (Retrieval) لربط المحرك بالوثائق المحلية ومنع الإجابات الخارجة عن السياق.
*   برمجة نظام إدارة الجلسات لضمان استمرارية الحوار وتدفق الإجابات (Streaming) بشكل لحظي.

### 3. التصميم (UI/UX)
*   تخصيص الواجهة عبر CSS لحذف الخلفيات الافتراضية واستبدالها بتصميم زجاجي احترافي.
*   ضبط صندوق السؤال بخلفية داكنة وإطار ذهبي، وتمييز التنبيهات بكلمة **تنبيه** وعبارة **كيف يمكنني مساعدتكم اليوم؟** باللون الأصفر العريض (**Bold Yellow**).

### 4. النشر والأمن (Deployment)
*   إدارة مفاتيح الـ API بشكل آمن عبر `st.secrets` لمنع الوصول غير المصرح به.
*   نشر التطبيق على سحابة Streamlit Cloud وربطه بمستودع GitHub للتحديث المستمر.

## 📖 التشغيل المحلي (Local Setup)

1. **تحميل المشروع:**
   ```bash

   cd apia-expert


تثبيت المكتبات:

Bash
pip install -r requirements.txt


3. **إعداد المفاتيح:**
   أنشئ ملفاً في المسار `.streamlit/secrets.toml` وأضف مفتاحك الخاص بـ Gemini:
   ```toml
   GEMINI_API_KEY = "YOUR_API_KEY_HERE"
   


تشغيل التطبيق:

Bash
streamlit run app.py
